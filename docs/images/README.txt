# Image Placeholders

This directory contains placeholder images referenced in the main README.md file.

The following images should be added to this directory:
- main-screen.png - Screenshot of the main application window
- login-screen.png - Screenshot of the login/register interface
- learn-words.png - Screenshot of the word learning interface
- translation.png - Screenshot of the translation tool
- leaderboard.png - Screenshot of the leaderboard functionality

When taking screenshots for the application, please:
1. Use a 16:9 aspect ratio when possible
2. Ensure that no sensitive or personal information is visible
3. Make sure the UI is in a clean state (no development overlays)
4. Use high-resolution images (at least 1920x1080)
5. Save in PNG format for maximum quality
